🔄 How to update your app later

Every time you push to GitHub, Render auto-redeploys. No manual steps needed.

# After making any change to your files: git add . git commit -m "update: describe your change" git push # Render detects the push and redeploys automatically ✓

🛠 Troubleshooting

❌ Build fails: "No module named sklearn"
Make sure scikit-learn is in your requirements.txt (not sklearn). The package name on PyPI is scikit-learn.
❌ "FileNotFoundError: adaboost_model.pkl"
You forgot to commit the pickle files. Run git add adaboost_model.pkl scaler.pkl then commit and push again.
⚠️ App sleeps after 15 minutes of inactivity (free tier)
Free Render services spin down when idle. The first request after sleep takes ~30 seconds to wake up. This is normal on the free plan.
⚠️ "Application failed to respond"
Check that your Start Command is exactly gunicorn wsgi:app --workers 2 --timeout 120 and that wsgi.py is in your repo root.
